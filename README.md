# Hanging360-AI-BOT (Claudia Ops AI)

AI + chatbot + operations manager for Hanging360 (home services scheduling like Uber), with:
- Multi-tenant, per-user persistent memory
- Tool-based operations (appointments, billing, reports, routing, pricing)
- Bilingual (EN/ES), default EN
- Audit logs for all mutations
